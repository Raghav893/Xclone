## 🚀 Getting Started

### Prerequisites
- Java 17+
- Maven 3.8+
- MySQL 8+

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/your-project.git

# Navigate into the project
cd your-project

# Install dependencies
mvn install

# Run the application
mvn spring-boot:run
```

The server will start at `http://localhost:8080`
